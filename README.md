## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- npm

### Setup

- Make sure you have npm set up

### Install

- Run npm i

### Usage

- Run npm run start

### Tests

- Run npm test

### Deployment

- The front-end can be deployed following the get started section on netlify
